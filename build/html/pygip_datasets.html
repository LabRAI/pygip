class pygip.datasets.datasets.Dataset(api_type='dgl', path='./downloads/')[source]¶

Bases: object

Base class for handling and loading GNN datasets in both DGL and PyG formats.

api_type¶

Specifies which API to use (‘dgl’ or ‘torch_geometric’).

Type:

str

path¶

The directory path where the dataset is located or will be downloaded.

Type:

str

dataset_name¶

The name of the dataset.

Type:

str

node_number¶

Total number of nodes in the graph.

Type:

int

feature_number¶

Dimensionality of the node features.

Type:

int

label_number¶

Number of distinct labels or classes.

Type:

int

features¶

Node features.

Type:

torch.Tensor

labels¶

Node labels.

Type:

torch.Tensor

train_mask¶

Boolean mask indicating which nodes are used for training.

Type:

torch.Tensor

test_mask¶

Boolean mask indicating which nodes are used for testing.

Type:

torch.Tensor

generate_train_test_masks()[source]¶

Generate train, validation, and test masks using the attribute ‘train_ratio’ (and a derived approach for val/test). By default, splits data randomly.

Note

Not all subclasses use this method. Some datasets inherently have train/val/test masks.

load_dgl_data()[source]¶

Load the dataset using DGL format.

Raises:

NotImplementedError – Must be implemented in subclass.

load_tg_data()[source]¶

Load the dataset using PyG format.

Raises:

NotImplementedError – Must be implemented in subclass.

pygip.datasets.datasets.dgl_to_tg(dgl_graph)[source]¶

Convert a DGLGraph to a PyG Data object.

Parameters:

dgl_graph (dgl.DGLGraph) – The input DGL graph to be converted.

Returns:

The resulting PyTorch Geometric Data object containing node features (x), edge information (edge_index), labels (y), and train/val/test masks.

Return type:

PyGData

pygip.datasets.datasets.tg_to_dgl(py_g_data)[source]¶

Parameters:
